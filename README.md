# AvantGarde-Villa-Booking-UI
4 farklı yapay zeka modeli (Prompt Engineering) kullanılarak geliştirilen avangart villa kiralama arayüzü. HTML. CSS ve Vanilla JS kullanılarak inşa edilmiştir. Dinamik veri filtreleme, DOM manipülasyonu, çoklu görsel modalları ve gerçek zamanlı fiyat hesaplama algoritmaları içeren responsive bir front-end projesi.
